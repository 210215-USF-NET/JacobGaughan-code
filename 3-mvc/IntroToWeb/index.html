Types of Selectors

Simple Selectors

  • Tag/Element
    • uses the tag name to identify the element
    • this will affect all elements with that tag name
  • Class
    • uses the class name to style a group of elements
    • denoted by .className
  • ID
    • uses a unique id
    • denoted by #id
  • Pseudo-class
    • specific keyword at the end of the selector used to style elements when it is at a certain state
    • :active, :hover, etc....
  • Pseudo-element
    • used to style specific parts of an element
    • insert content before or after an element
    • style first letter or line of an element
    • ::before, ::after, ::first-line, ::first-letter

Combinator Selectors

  • Child Descendant
  • Sibling
  • Attribute selectors
  • Multiple selectors

Wild card selectors

  • just the *
  • Styles all the elements
